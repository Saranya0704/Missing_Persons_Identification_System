# Missing Persons Identification System

A face recognition-based missing persons tracking system built with Streamlit and DeepFace.

## Features
- 📋 File a missing person report with photo
- 🔍 Identify a person using face recognition
- ✅ View matched/found cases with location
- 📊 Dashboard with live statistics

## Tech Stack
- Python
- Streamlit
- DeepFace (Facenet512)
- SQLite
- OpenCV

## How to Run Locally
pip install -r requirements.txt
streamlit run Dashboard.py
